Welcome to my portofolio!

Vasile Hij - Python Django developer

I build backend systems with Python and Django, designing RESTful APIs, data pipelines with Pandas and Airflow, and relational schemas on MySQL. On the frontend I work with Vue.js and vanilla JavaScript, keeping the interface functional and responsive without over-engineering it.

On the infrastructure side I containerize workloads with Docker, orchestrate them on Kubernetes (K8s and K3s), and deploy to AWS and Hetzner from a Linux-first environment. I manage source control through Git with GitHub and Bitbucket, track work in Jira and Confluence, and validate APIs with Postman before they ship.

Beyond the day job I run a home lab where I self-host services on Proxmox, manage storage with TrueNAS and Synology, automate the network with Omada, and prototype IoT devices using ESP32 and Arduino under Home Assistant. Golang and C++ come in handy at the lower levels of that stack.
